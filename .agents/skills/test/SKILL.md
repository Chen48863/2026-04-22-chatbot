# Test（測試驗證）Skill

## 描述

此技能用於生成手動測試清單（Manual Test Checklist）。當使用者要求建立測試計畫或驗證功能時，AI Agent 應依照以下結構產出完整的手動測試清單文件，確保每個功能皆有對應的測試案例可供驗證。

## 觸發條件

當使用者提出以下類型的請求時啟用此技能：
- 「/test」
- 「幫我寫測試清單」
- 「產生測試計畫」
- 「建立手動測試案例」
- 「驗證功能」
- 「產出測試文件」

## 產出文件

此技能產出一份 Markdown 格式的**手動測試清單文件**，檔名建議為 `test_checklist.md`。

## 注意事項

- 所有測試清單內容應以**繁體中文**撰寫
- 測試案例應根據實際專案的功能進行**客製化調整**
- 若 PRD 文件已存在，測試案例應覆蓋 PRD 中列出的所有**功能需求**
- 若 Implement 文件已存在，測試步驟應對應實際的**路由與頁面**
- 每個測試案例的步驟應足夠具體，讓**非開發人員**也能執行
- 「實際結果」與「通過」欄位留空，由測試人員在執行測試時填寫
- 測試清單應保持**活文件（Living Document）**的精神，隨功能變更持續更新
