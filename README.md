# 🤖 智慧聊天機器人（Smart Chatbot）

一個基於 **FastAPI + SQLite** 的智慧聊天機器人 Web 應用程式，支援多輪對話、多聊天室管理、檔案上傳與記憶機制。

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 💬 多輪對話 | 支援建立多個聊天室，每個聊天室維持獨立的對話上下文 |
| 📁 檔案上傳 | 支援上傳圖片（jpg/png/gif）與文件（pdf/txt/docx），單檔上限 10MB |
| 🧠 記憶機制 | 儲存使用者偏好（顯示名稱、語言、回覆風格），跨聊天室持續有效 |
| 🔄 重新生成 | 可重新產生 AI 的最後一則回覆 |
| ✏️ 聊天室管理 | 新增 / 刪除 / 重新命名聊天室 |
| ⚙️ 設定面板 | 可自訂顯示名稱、偏好語言與回覆風格 |
| 📱 響應式設計 | 支援桌面與行動裝置瀏覽 |

## 🏗️ 技術架構

```
┌─────────────────────────────────────────────────┐
│                   前端 (Frontend)                 │
│         HTML + CSS + JavaScript (Jinja2)          │
├─────────────────────────────────────────────────┤
│                   後端 (Backend)                  │
│              FastAPI (Python 非同步框架)            │
├─────────────────────────────────────────────────┤
│                  資料庫 (Database)                 │
│             SQLite + SQLAlchemy ORM               │
├─────────────────────────────────────────────────┤
│                  AI 回覆模組                       │
│         Mock 模擬回應（預留 AI API 串接介面）        │
└─────────────────────────────────────────────────┘
```

- **前端**：HTML + CSS + JavaScript，使用 Jinja2 模板引擎
- **後端**：FastAPI（Python 非同步 Web 框架）
- **資料庫**：SQLite（透過 SQLAlchemy ORM 操作）
- **AI 回覆**：Mock 模擬回應（可日後替換為 Google Gemini / OpenAI API）

## 🚀 安裝與執行

### 前置需求

- **Python 3.10+**（建議 3.11 以上）
- **pip** 套件管理工具

### 安裝步驟

1. **進入專案目錄**
   ```bash
   cd 20260422
   ```

2. **安裝依賴套件**
   ```bash
   pip install -r requirements.txt
   ```
   或手動安裝：
   ```bash
   pip install fastapi uvicorn[standard] sqlalchemy jinja2 python-multipart aiofiles
   ```

3. **啟動應用程式**
   ```bash
   python app.py
   ```

4. **開啟瀏覽器**

   前往 👉 **http://127.0.0.1:8000**

> 💡 伺服器啟動後會自動建立 `database.db` 資料庫與 `uploads/` 目錄。
> 按 `Ctrl+C` 可停止伺服器。

## 📖 使用說明

### 基本操作

1. **建立對話**：點擊左側邊欄的「＋」按鈕建立新聊天室
2. **發送訊息**：在輸入框輸入文字，按 `Enter` 發送（`Shift+Enter` 換行）
3. **上傳檔案**：點擊輸入框旁的 📎 按鈕選擇檔案上傳
4. **切換對話**：點擊左側邊欄中的聊天室名稱進行切換
5. **重新生成**：對 AI 回覆不滿意時，可點擊重新生成按鈕

### 聊天室管理

- **重新命名**：右鍵點擊聊天室名稱可重新命名
- **刪除對話**：點擊聊天室的刪除按鈕可移除整個聊天室

### 偏好設定

點擊左下角「⚙️ 設定」按鈕，可自訂：
- **顯示名稱**：機器人會用您的名稱稱呼您
- **偏好語言**：繁體中文 / English / 日本語
- **回覆風格**：友善 / 正式 / 簡潔 / 詳細

### Mock AI 可辨識的問題

| 觸發關鍵字 | 範例 | 回覆內容 |
|-----------|------|---------|
| `你好`、`哈囉` | 「你好啊！」 | 友善的問候語 |
| `名字`、`你是誰` | 「你叫什麼名字？」 | 自我介紹並稱呼使用者 |
| `謝謝`、`感謝` | 「謝謝你！」 | 禮貌回覆 |
| `再見`、`掰掰` | 「掰掰～」 | 道別語 |
| 其他任意輸入 | 任何問題 | 隨機選擇預設回覆 |

## 📡 API 文件

啟動伺服器後，可存取自動產生的 API 文件：

- **Swagger UI**：http://127.0.0.1:8000/docs
- **ReDoc**：http://127.0.0.1:8000/redoc

### 主要 API 端點

| 方法 | 路徑 | 說明 |
|------|------|------|
| `GET` | `/api/sessions` | 取得所有聊天室列表 |
| `POST` | `/api/sessions` | 建立新聊天室 |
| `GET` | `/api/sessions/{id}` | 取得聊天室詳情與訊息 |
| `PUT` | `/api/sessions/{id}` | 更新聊天室標題 |
| `DELETE` | `/api/sessions/{id}` | 刪除聊天室 |
| `POST` | `/api/sessions/{id}/messages` | 發送訊息並取得 AI 回覆 |
| `POST` | `/api/sessions/{id}/regenerate` | 重新生成最後一則 AI 回覆 |
| `POST` | `/api/upload` | 上傳檔案 |
| `GET` | `/api/memory` | 取得所有使用者記憶 |
| `PUT` | `/api/memory` | 更新使用者記憶 |
